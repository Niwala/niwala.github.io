# Flora Object
Asset configurable par l'utilisateur.

## Implémentations
- [Flora plant](api\flora-objects\flora-plant.md)
- [Flora pack](api\flora-objects\flora-pack.md)

Un FloraScriptableObject n'a pas pour objectifs de générer directement de la géométrie. Son seul objectif est de pouvoir fournir des [Flora Generation Process](api\generation\flora-generation-process.md) en fonction de ses paramètres.
