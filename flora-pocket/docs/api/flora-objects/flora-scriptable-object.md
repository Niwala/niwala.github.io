# Flora Scriptable Object
Asset configurable par l'utilisateur.

## Implémentations
- [api-flora-plant]
- [api-flora-pack]
- [api-flora-creeper]

Un FloraScriptableObject n'a pas pour objectifs de générer directement de la géométrie. Son seul objectif est de pouvoir fournir des [Flora Generation Process](api/api-generation/api-flora-generation-process.md) en fonction de ses paramètres.
