# Flora Scriptable Object
Asset configurable par l'utilisateur.

## Implémentations
- FloraPlant
- FloraPack
- FloraCreeper

Un FloraScriptableObject n'a pas pour objectifs de générer directement de la géométrie. Son seul objectif est de pouvoir fournir des [Flora Generation Process](Flora-generation-process) en fonction de ses paramètres.
