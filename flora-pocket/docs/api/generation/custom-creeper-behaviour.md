
# Custom creeper behaviour
Il est possible de créer votre propre méthode de croissance sur les creepers.
Cela vous permet de créer un creeper avec un comportement spécifique tout en bénéficiant de l'écosystème déjà mis en place.


## Votre classe custom
Votre classe doit se déclarer comme ceci : 
```csharp
public class MyCustomCreeperExecution : FloraExecution<Creeper>
{
	//


}
```
